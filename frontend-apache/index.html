Notas operativas

  • Si la API y esta web están en distintos hosts o puertos, la API debe permitir CORS.
  • Para despliegue en Apache basta con copiar index.html, styles.css, config.js y app.js al DocumentRoot.
  • Validar desde navegador y también con curl sobre la API.
